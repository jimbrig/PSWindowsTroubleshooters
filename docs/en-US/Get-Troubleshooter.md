# Get-Troubleshooter

## SYNOPSIS
Get-Troubleshooter is a function that returns a list of troubleshooters that are available on the local machine.

## SYNTAX

```
Get-Troubleshooter [[-Name] <String[]>] [-All] [<CommonParameters>]
```

## DESCRIPTION
Get-Troubleshooter is a function that returns a list of troubleshooters that are available on the local machine.

## EXAMPLES

### EXAMPLE 1
```
Get-Troubleshooter
```
